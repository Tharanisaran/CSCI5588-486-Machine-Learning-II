How to run the code for Hillclimbing algorithm

Open Command Prompt 

Traverse to the folder PA1_Tharani_Maaneeivaannan_2575198/Executable
Then run hillclimbing.exe > hillclimbing_output.txt

The output can be verified by opening the hillclimbing_output.txt
